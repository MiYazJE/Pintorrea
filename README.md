## INSTALACIÓN
Para correr este proyecto localmente debes tener instalado:
+ Mongodb
+ Node js

Ejecutar por consola en la raíz del proyecto:
```
$ npm i
$ npm install client
$ npm run dev
$ npm run client
$ Ir a http://localhost:3000
```
